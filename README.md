# Rust_Encryption
Rustで書かれたシンプルなファイル暗号化・復号ツール。

## Description
このプロジェクトは、Rustの学習とファイルI/O処理の実践を目的とした制作物です。
強力な暗号化アルゴリズム（未定）を使用して、個人のファイルを簡単に保護できます。
複雑な設定なしで初心者でもすぐに使えるソフトウェアを目指しました。

## Usage
特定のテキストファイルを暗号化します。本ツールは特定の形式に限定せず、あらゆるバイナリファイルを対象にできます。
例：text.txtを入力して、暗号化したファイルはtext.txt.xxx(未定)あるいはtext.xxx(未定)
