# Rust_Encryption
Rustで書かれたシンプルなファイル暗号化・復号ツール。

## Description
このプロジェクトは、Rustの学習とファイルI/O処理の実践を目的とした制作物です。
強力な暗号化アルゴリズム（未定）を使用して、個人のファイルを簡単に保護できます。
複雑な設定なしで初心者でもすぐに使えるソフトウェアを目指しました。

## Usage
以下のコマンドを実行してヘルプを表示します：

rust_encryption --help

ファイルの暗号化：

rust_encryption encrypt file_path

ファイルの復号：

rust_encryption decrypt file_path
