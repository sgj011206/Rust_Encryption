WHY RUST ENCRYPTION

シンプルでも、セキュリティに妥協しない

分かりやすいコマンドラインの流れに主要操作をまとめ、初心者がRustを実践しながら確実にファイルを保護できるようにしています。

01

AES-256-GCM

現代的な認証付き暗号を使用し、データの機密性を守りながら完全性も検証します。

02

シンプルなコマンド

直感的な encrypt、decrypt、keygen コマンドで主要な操作を実行できます。

03

Rustで実装

Rust学習とファイルI/Oの実践を軸に、性能、保守性、メモリ安全性を両立しています。

04

オープンで透明

MITライセンスで公開されており、自由に学習・変更・拡張できます。

SECURITY WARNING

鍵はデータにアクセスする唯一の手段です

鍵は必ず安全で信頼でき、バックアップのある場所に保管してください。鍵を失うと、暗号化されたデータは復元できません。

鍵と暗号化ファイルを同じ場所に保存しない 重要な鍵はオフラインでバックアップする 信頼できない経路で鍵を送信しない
ABOUT THE PROJECT

実用的なツールでRustを学ぶ

Rust Encryptionは学習と実践を目的としたオープンソースプロジェクトで、Rustの基礎、ファイルI/O、暗号化フローに焦点を当てています。利用のハードルを低く抑え、初心者でも複雑な設定なしでファイル暗号化ツールを理解し、使用できることを目指しています。
