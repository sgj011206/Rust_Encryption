WHY RUST ENCRYPTION

简单，不代表妥协安全

项目以清晰的命令行工作流封装核心操作，帮助初学者在实践 Rust 的同时完成可靠的文件保护。

01

AES-256-GCM

使用现代认证加密算法，在保护数据机密性的同时验证数据完整性。

02

极简命令

通过直观的 encrypt、decrypt 和 keygen 完成核心操作。

03

Rust 实现

围绕 Rust 学习与文件 I/O 实践构建，兼顾性能、可维护性与内存安全。

04

SECURITY WARNING

密钥就是数据的唯一入口

请务必将密钥保存在安全、可靠且有备份的位置。一旦密钥丢失，加密数据将无法恢复。

不要将密钥与加密文件放在同一位置 建议为重要密钥建立离线备份 不要通过不可信渠道传输密钥
ABOUT THE PROJECT

在真实工具中学习 Rust

Rust Encryption 是一个面向学习与实践的开源项目，聚焦 Rust 基础、文件 I/O 与加密流程。它追求较低的使用门槛，让初学者无需复杂配置，也能理解并使用文件加密工具。
